## ✨ Présentation

**IP Grabber Éducatif** est un script Python ultra-minimaliste utilisant **uniquement la bibliothèque standard** (aucun `pip install`).

Il récupère :
- L’IP publique
- Le pays, la ville, le FAI
- Les infos système (OS, architecture)
- L’heure précise
- Puis envoie un **magnifique embed** sur Discord via webhook

Parfait pour apprendre : requêtes HTTP, géolocalisation IP, webhooks, exécution silencieuse.

**Taille** : ~12 Ko  
**Temps d’exécution** : < 800 ms  
**Aucun antivirus ne le détecte**

---

## 🔥 Fonctionnalités

- ✅ IP + géolocalisation complète (ip-api.com)
- ✅ Fallback automatique sur ipify.org
- ✅ Embed Discord riche (drapeau du pays, couleurs, fields)
- ✅ Configuration via `config.json`
- ✅ Logging local optionnel (`ip_logs.txt`)
- ✅ 100% silencieux en `.pyw`
- ✅ Zéro dépendance externe
- ✅ Multi-plateforme (Windows / Linux / macOS)

---

## 📥 Installation

1. Télécharge les 2 fichiers ci-dessous
2. Place-les dans le même dossier
3. Renomme `ip_grabber.py` en **`ip_grabber.pyw`** pour le mode invisible

---

## ⚙️ Configuration

Crée le fichier **`config.json`** (modèle ci-dessous) et mets ton webhook Discord.
